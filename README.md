# wantedbot

Ein einfacher WhatsApp Bot mit Baileys.

🔄 Bot neu verbinden

Wenn der Bot neu verbunden werden soll:

rm -r ./auth

▶️ Bot starten

node index.js

⚙️ Prefix

📌 Standard Prefix: .

🔁 Prefix zurücksetzen

node resetprefix.js

📦 Installation

npm install

🚀 Kurzbefehle

Aktion

Befehl

Bot starten

node index.js

Auth reset

rm -r ./auth

Prefix reset

node resetprefix.js

⚠️ Hinweis

Dieses Projekt ist für Lern- und Testzwecke gedacht.
